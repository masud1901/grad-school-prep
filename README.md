# Prof Ning Lu Prep

Personal study materials and course prep: Computer Networks, Math for ML/DS, and Convex Optimization.

## Contents

| Folder | Description |
|--------|-------------|
| **Computer Networks** | CS-style fundamentals: Tanenbaum textbook, lectures, and notes (protocols, TCP/IP, wireless). See `Computer Networks/Fundamentals/CLAUDE.md` for the study guide. |
| **math for ML and DS** | Linear algebra for ML (DeepLearning.AI–style): assignments (Jupyter), solutions, and graded quiz answers. Topics: linear transformations, eigenvalues, PCA. |
| **Convex Optimization** | Convex optimization reference (e.g. Boyd book PDF). |

## Structure

- **PDFs**: Lectures, textbook chapters, and handouts are kept in the repo.
- **Markdown**: `CLAUDE.md` is the main study guide for the networks material; other `.md` files are solutions and quiz answers.
- **Notebooks**: `C1W2`–`C1W4` Jupyter assignments in `math for ML and DS/Linear transfromations/`.

## Use

Clone and use locally. No setup required beyond Python + Jupyter if you want to run the notebooks. PDFs and markdown are for reading and study.
